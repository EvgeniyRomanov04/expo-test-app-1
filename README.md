# 📱 Mecenate Feed App

Тестовое задание: реализация экрана ленты публикаций для платформы **Mecenate**

---

## 🚀 Описание проекта

Приложение представляет собой экран **Feed (ленты постов)**, где пользователь видит публикации авторов, на которых он подписан.

### 📦 Функциональность:

#### 📰 Экран Feed (лента)

- 📃 Список постов (аватар автора, имя, текст-превью, обложка)
- 🗂️ Таб-фильтр: **Все / Бесплатные / Платные**
- ❤️ Счётчики лайков и комментариев
- 🔄 Pull-to-refresh
- 📥 Курсорная пагинация (подгрузка при скролле)
- 🔒 Закрытые посты (`tier: paid`) отображаются с заглушкой
- 👆 Tap по посту → переход на детальный экран
- ❌ Обработка ошибок API + кнопка повтора

#### 📄 Экран Post Detail

- 📝 Полный текст публикации
- 🖼️ Обложка поста
- 👤 Имя автора
- ❤️ Кнопка лайка с анимацией (Reanimated)
- 📳 Haptic feedback при лайке
- 💬 Список комментариев с lazy load
- ✍️ Поле ввода + отправка нового комментария
- ⚡ Real-time обновление лайков и комментариев через WebSocket

## 🛠️ Установка и запуск

### 1. Установка зависимостей

```bash
npm install
```

### 2. Автоматическая postinstall настройка

```bash
npm run postinstall
```

#### 📌 Скрипт:

- и справляет конфигурацию React Native Reanimated
- подготавливает native зависимости

### 3. Prebuild Expo

```bash
npx expo prebuild
```

#### 📌 Это нужно для:

- генерации native проектов (iOS / Android)
- корректной работы Reanimated и native модулей

### 4. Запуск приложения!

```bash
npx expo start
```

### ⚙️ Возможные проблемы

Если что-то сломалось:

```bash
npx expo start -c
```

или повторно:

```bash
rm -rf node_modules && npm install
```
